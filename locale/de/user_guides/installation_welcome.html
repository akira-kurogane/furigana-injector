Jetzt ist Ihr Firefox Browser in der Lage, Furigana einzufügen, die kleinen Hiragana Buchstaben über oder neben schweren Kanji Worten. Versuchen Sie es - klicken sie auf das -Symbol in der Statusleiste.

Was sollte ich zuerst tun?

Installieren Sie die XHTML Ruby Support Erweiterung

(Link auf die Seite von mozilla.org fur Shimoda Hiroshi's Erweiterung.)

Mit Ruby-Support:
とうきょうおおさか
東京から大阪までの...
Ohne Ruby-Support: 東京（とうきょう）から大阪（おおさか）までの...

Wählen Sie ein Kanji-Lese-Niveau

Wählen Sie die Kanji, für die Sie keine Furigana anzeigen lassen wollen.

Setzen Sie Ihre Kanji-Lesung Ebene

Verwenden Sie die "Optionen ..." (siehe unten), um eine Liste der Kanji Sie brauchen nicht zu Furigana. Hinweis: Wenn Sie bereits installiert haben die XHTML Ruby Support Addon gibt es eine einfache und schnelle Auswahl von Kanji-Seite, die Sie verwenden können, statt.


Dann nutzen Sie dieses Tool auf jeder beliebigen Web-Seite mit japanischem Text. www.mozilla.gr.jp ist ein gutes Beispiel, das zeigt:

  • Worte aus einfachen Kanji werden überhaupt nicht verarbeitet.
  • Worte in Links werden nicht verarbeitet, es sei denn, Sie passen die entsprechende Einstellung an.
  • Bilder werden ignoriert, daher wird Text in Grafiken nicht beeinflusst, sorry.

Datenschutz-Hinweis

Bitte beachten Sie, dass ab der Version 2.0 der Text auf einem externen Server verarbeitet wird. Verwenden Sie Furigana Injector bitte nicht bei Webseiten oder Absätzen oder Sätzen, die persönlichen Daten enthalten.


Andere Optionen ...

  • Sie können wählen, ob Furigana zusatzlich zu normalem Text auch Hyperlinks hinzugefügt werden sollen.

Auf diese Optionen kann direkt durch einen rechten Mausklick auf das -Symbol zugegriffen werden.
