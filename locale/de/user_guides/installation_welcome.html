Jetzt ist Ihr Firefox Browser in der Lage, Furigana einzufugen, die kleinen Hiragana Buchstaben uber oder neben schweren Kanji Wortern. Versuchen Sie es - klicken sie auf das -Symbol in der Statusleiste.

Was sollte ich zuerst tun?

Installieren Sie die XHMTL Ruby Support Erweiterung

(Link auf die Seite von mozilla.org fur Shimoda Hiroshi's Erweiterung.)

Mit Ruby-Support: 東京(とうきょう) から大阪(おおさか) までの...
Ohne Ruby-Support: 東京（とうきょう）から大阪（おおさか）までの...

Wahlen Sie ein Kanji-Lesungs-Niveau

Wahlen Sie die Kanji, fur die Sie keine Furigana anzeigen lassen wollen.


Dann nutzen Sie dieses Tool auf jeder beliebigen Web-Seite mit japanischem Text. www.mozilla.gr.jp ist ein gutes Beispiel, das zeigt:

  • Worte aus einfachen Kanji werden uberhaupt nicht verarbeitet.
  • Worte in Links werden nicht verarbeitet, es sei denn, Sie passen die entsprechende Einstellung an.
  • Bilder werden ignoriert, daher wird Text in Grafiken nicht beeinflusst, sorry.


Andere Optionen ...

  • Sie konnen wahlen, ob alle Seiten direkt beim Laden mit Furigana versehen werden sollen.
  • Sie konnen wahlen, ob Furigana zusatzlich zu normalem Text auch Hyperlinks hinzugefugt werden.

Auf diese Optionen kann direkt durch einen rechten Mausklick auf das -Symbol zugegriffen werden.
