What should I do first?

Install the XHMTL Ruby support extension

(Link to mozilla.org's page for Shimoda Hiroshi's add-on.)

With ruby support:
とうきょうおおさか
東京から大阪までの...
Without ruby support: 東京（とうきょう）から大阪（おおさか）までの...

Install the Mecab package

The Mecab package is required for Furigana Injector. Please download the relevant version.

  • Windows Choose the "mecab-win32" download. Japanese Windows users can use either Shift-JIS and UTF-8 dictionary. Others should only install the UTF-8 version.
  • Linux If the "mecab" and "mecab-ipadic" rpm packages are available for your distribution simply install them. If that's not possible download the "mecab" and "mecab-ipadic" source packages and build them. Please "./configure --with-charset=utf8" if building the mecab-ipadic dictionary package from source.
  • Mac OSX ???? Download the "mecab" and "mecab-ipadic" source packages and build them??? Please "./configure --with-charset=utf8" if building the mecab-ipadic dictionary package from source.

Select your kanji-reading level

Choose the kanji you do and don't want furigana for.

Set your kanji-reading level

Use the "Options..." (see below) to set the list of the kanji you don't need furigana for. Note: if you have already installed the XHTML Ruby Support addon there is a quick and easy Kanji-selecting page you can use instead.


Then use this tool in any web page that has japanese text. www.mozilla.gr.jp is a good example that shows:

  • Words composed of simple kanji won't be processed at all.
  • Words in links wont be processed, unless you change the preference for that.
  • Images are ignored so text in graphics won't be changed, sorry.


Other options ...

  • You can choose to have all pages processed for furigana as soon as they've loaded.
  • You can choose to add furigana to text in hyperlinks as well as normal text.

These options can be accessed directly by right-clicking on the icon.
