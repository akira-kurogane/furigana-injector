Maintenant votre navigateur Firefox a la possibilité d'injecter des furigana, ces petits caractères en hiragana au dessus ou à côté des kanji difficiles. Essayez, il suffit de cliquer sur l'icône dans la barre de status.

Que faire en premier ?

Installer l'extension Support de XHTML Ruby

(Link to mozilla.org's page for Shimoda Hiroshi's add-on.)

Avec le support de ruby : 東京(とうきょう) から大阪(おおさか) までの...
Sans le support de ruby : 東京（とうきょう）から大阪（おおさか）までの...

Selectionner votre niveau de lecture des kanji

Choisissez les kanji pour lesquels vous désirez des furigana ou pas.


Ensuite, utilisez cet outil dans n'importe quelle page affichant du texte japonais. Le site www.mozilla.gr.jp est un bon exemple qui montre :

  • Des mots composés de kanji simples, qui ne seront pas traités.
  • Des mots à l'intérieur des liens, qui ne seront pas traités, à moins que vous ne changiez cette option.
  • Des images qui seront ignorées, donc leur texte ne sera pas modifié, désolé.


Autres options ...

  • Vous pouvez choisir d'avoir toutes les pages traitées, afin qu'elles affichent les furigana une fois chargées.
  • Vous pouvez choisir d'injecter les furigana également dans les textes à l'intérieur des liens.

Ces options sont directement accessibles en faisant un click droit sur l'icône dans la barre de status.
