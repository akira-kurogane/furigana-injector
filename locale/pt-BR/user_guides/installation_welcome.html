O que devo fazer primeiro?

Instalar a extensão XHMTL Ruby support

Selecionar seu nível de leitura de kanji

Para configurar quais kanji você quer e quais não quer o furigana.

Configure seu nível de leitura de kanji

Use "Opções..." (veja abaixo) para configurar a lista dos kanji que você não precisa de furigana. Nota: se você já instalou a extensão do XHTML Ruby Support, tem uma página rápida e fácil para configurar os kanji que você quer.


Então ative essa extensão em qualquer página com textos em japonês. Um bom exemplo é o site www.mozilla.gr.jp, que mostra:

  • Que palavras com kanji simples não serão processadas.
  • Que palavras em links não serão processadas, a não ser que você configure isso.
  • Que textos em imagens não terão furigana, desculpe.


Outras opções ...

  • Você pode selecionar para que todas as páginas já carreguem com furigana.
  • Você pode selecionar para adicionar também furigana em links.

Essas opções podem ser acessadas diretamente clicando com o botão direito no ícone na barra de status.
